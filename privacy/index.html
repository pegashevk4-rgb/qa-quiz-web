Privacy

Политика конфиденциальности

QA Hiring Assistant — сервис для оценки QA-кандидатов. Мы собираем только те данные, которые нужны для прохождения тестов, формирования отчётов и работы HR-дашборда.

Какие данные мы можем собирать

  • Имя кандидата;
  • Email кандидата;
  • Ответы на тестовые вопросы;
  • Результаты и аналитика тестирования;
  • Технические данные браузера для стабильной работы сервиса.

Как используются данные

Данные используются только для:

  • прохождения QA-тестов;
  • формирования отчётов для HR и тимлидов;
  • аналитики результатов кандидатов;
  • улучшения качества сервиса.

Мы не продаём и не передаём персональные данные третьим лицам.

Хранение данных

Результаты тестов и связанные данные могут храниться на серверах сервиса для доступа HR-специалистов к отчётам.

При необходимости данные могут быть удалены по запросу.

Cookies

Сервис может использовать cookies в будущем для сохранения настроек интерфейса и авторизации.
